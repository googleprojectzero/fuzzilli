Checks if the given wide character corresponds (if narrowed) to one of the ten decimal digit characters 0123456789 . If the value of ch is neither representable as a wchar_t nor equal to the value of the macro WEOF , the behavior is undefined. Contents 1 Parameters 2 Return value 3 Notes 4 Example 5 See also [ edit ] Parameters ch - wide character [ edit ] Return value Non-zero value if the wide character is a numeric character, zero otherwise. [ edit ] Notes std::iswdigit and std::iswxdigit are the only standard wide character classification functions that are not affected by the currently installed C locale.
