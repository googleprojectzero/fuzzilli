Checks if the given character is a control character as classified by the currently installed C locale. In the default, "C" locale, the control characters are the characters with the codes 0x00-0x1F and 0x7F . The behavior is undefined if the value of ch is not representable as unsigned char and is not equal to EOF . [ edit ] Notes Like all other functions from <cctype> , the behavior of std::iscntrl is undefined if the argument's value is neither representable as unsigned char nor equal to EOF .
