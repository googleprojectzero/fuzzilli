std:: tm * gmtime ( const std:: time_t * time ) ; Converts given time since epoch as std::time_t value into calendar time, expressed in Coordinated Universal Time (UTC). Contents 1 Parameters 2 Return value 3 Notes 4 Example 5 See also [ edit ] Parameters time - pointer to a time_t object to convert [ edit ] Return value Pointer to a static internal std::tm object on success, or null pointer otherwise. The structure may be shared between std::gmtime , std::localtime , and std::ctime and may be overwritten on each invocation. [ edit ] Notes This function may not be thread-safe. POSIX requires that gmtime sets errno to EOVERFLOW if it fails because the argument is too large.
