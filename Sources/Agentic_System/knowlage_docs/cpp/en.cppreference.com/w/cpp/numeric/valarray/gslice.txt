Objects of type std::gslice can be used as indices with valarray's operator [ ] to select, for example, columns of a multidimensional array represented as a valarray . Given the starting value s , a list of strides i j and a list of sizes d j , a std::gslice constructed from these values selects the set of indices k j =s+Σ j (i j d j ) . For example, a gslice with starting index 3 , strides {19,4,1 } and lengths {2,4,3 } generates the following set of 24=2*4*3 indices: 3 + 0 * 19 + 0 * 4 + 0 * 1 = 3 , 3 + 0 * 19 + 0 * 4 + 1 * 1 = 4 , 3 + 0 * 19 + 0 * 4 + 2 * 1 = 5 , 3 + 0 * 19 + 1 * 4 + 0 * 1 = 7 , 3 + 0 * 19 + 1 * 4 + 1 * 1 = 8 , 3 + 0 * 19 + 1 * 4 + 2 * 1 = 9 , 3 + 0 * 19 + 2 * 4 + 0 * 1 = 11 , ... 3 + 1 * 19 + 3 * 4 + 1 * 1 = 35 , 3 + 1 * 19 + 3 * 4 + 2 * 1 = 36 It is possible to construct std::gslice objects that select some indices more than once: if the above example used the strides {1,1,1} , the indices would have been {3, 4, 5, 4, 5, 6, ...} . Such gslices may only be used as arguments to the const version of std::valarray::operator[] , otherwise the behavior is undefined.
