operator>> (C++11) (C++11) [edit] Defined in header <random> template < class IntType = int > class binomial_distribution ; (since C++11) Produces random non-negative integer values i , distributed according to discrete probability function: \(P(i|t,p) = \binom{t}{i} \cdot p^i \cdot (1-p)^{t-i}\) P(i|t,p) = ⎛ ⎜ ⎝ t i ⎞ ⎟ ⎠ · p i · (1 − p) t−i The value obtained is the number of successes in a sequence of t yes/no experiments, each of which succeeds with probability p . std::binomial_distribution satisfies RandomNumberDistribution .
