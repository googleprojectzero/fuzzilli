A) The square of f . B) The square of i . [ edit ] Notes The norm calculated by this function is also known as field norm or absolute square . The Euclidean norm of a complex number is provided by std::abs , which is more costly to compute. In some situations, it may be replaced by std::norm , for example, if abs ( z1 ) > abs ( z2 ) then norm ( z1 ) > norm ( z2 ) . The additional overloads are not required to be provided exactly as (A,B) . They only need to be sufficient to ensure that for their argument num : If num has a standard (until C++23) floating-point type T , then std :: norm ( num ) has the same effect as std :: norm ( std:: complex < T > ( num ) ) . Otherwise, if num has an integer type, then std :: norm ( num ) has the same effect as std :: norm ( std:: complex < double > ( num ) ) .
