program that instantiates a primary template of a mathematical constant variable template is ill-formed. The standard library specializes mathematical constant variable templates for all floating-point types (i.e. float , double , long double , and fixed width floating-point types (since C++23) ). A program may partially or explicitly specialize a mathematical constant variable template provided that the specialization depends on a program-defined type .
