std:: noop_coroutine_handle noop_coroutine ( ) noexcept ; (since C++20) Returns a coroutine handle referring to a no-op coroutine. If there has been already a coroutine state of no-op coroutine, it is unspecified whether a subsequent call to noop_coroutine returns a previously obtained coroutine handle, or a coroutine handle referring to a new coroutine state of no-op coroutine. [ edit ] Notes Return values from different calls to noop_coroutine may and may not compare equal. noop_coroutine may only return a noop_coroutine_handle referring to a coroutine state object without starting a coroutine.
