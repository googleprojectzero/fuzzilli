std:: chrono :: time_point < /*trivial-clock*/ > ; (since C++17) (until C++20) using file_time_type = std:: chrono :: time_point < std:: chrono :: file_clock > ; (since C++20) Represents file time. /*trivial-clock*/ is an implementation-defined type that satisfies TrivialClock and is sufficient to represent the resolution and range of the file time values offered by the filesystem.
