error_code & ec ) noexcept ; (2) (since C++17) Copies a symlink to another location. 1) Effectively calls f ( read_symlink ( from ) , to ) where f is create_symlink or create_directory_symlink depending on whether from resolves to a file or directory. 2) Effectively calls f ( read_symlink ( from, ec ) , to, ec ) where f is create_symlink or create_directory_symlink depending on whether from
