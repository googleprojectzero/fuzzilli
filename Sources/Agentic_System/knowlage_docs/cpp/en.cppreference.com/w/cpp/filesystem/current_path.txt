native format) by POSIX getcwd . (2) returns path ( ) if error occurs. 3,4) Changes the current working directory to p , as if by POSIX chdir . [ edit ] Notes The current working directory is the directory, associated with the process, that is used as the starting location in pathname resolution for relative paths. The current path as returned by many operating systems is a dangerous global variable. It may be changed unexpectedly by third-party or system library functions, or by another thread.
