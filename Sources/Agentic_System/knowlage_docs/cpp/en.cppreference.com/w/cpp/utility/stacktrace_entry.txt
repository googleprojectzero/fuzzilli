formatter <std::stacktrace_entry> [edit] Defined in header <stacktrace> class stacktrace_entry ; (since C++23) The stacktrace_entry class provides operations for querying information about an evaluation in a stacktrace. Each stacktrace_entry object is either empty, or represents an evaluation in a stacktrace. stacktrace_entry models std:: regular and std:: three_way_comparable < std:: strong_ordering > .
