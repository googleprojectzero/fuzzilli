Helper objects variant_npos [edit] Defined in header <variant> Defined in header <utility> (since C++26) struct monostate { } ; (since C++17) Unit type intended for use as a well-behaved empty alternative in std::variant . In particular, a variant of non-default-constructible types may list std::monostate as its first alternative: this makes the variant itself default-constructible.
