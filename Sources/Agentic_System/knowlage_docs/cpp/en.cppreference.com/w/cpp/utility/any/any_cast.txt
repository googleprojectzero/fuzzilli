Let U be std:: remove_cv_t < std:: remove_reference_t < T >> . 1) The program is ill-formed if std:: is_constructible_v < T, const U & > is false . 2) The program is ill-formed if std:: is_constructible_v < T, U & > is false . 3) The program is ill-formed if std:: is_constructible_v < T, U > is false . 4,5) The program is ill-formed if std:: is_void_v < T > is true . [ edit ] Exceptions 1-3) Throws std::bad_any_cast if the typeid of the requested T does not match that of the contents of operand .
