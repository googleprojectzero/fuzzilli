has_facet ( const locale & loc ) throw ( ) ; (until C++11) template < class Facet > bool has_facet ( const locale & loc ) noexcept ; (since C++11) Checks if the locale loc implements the facet Facet . The program is ill-formed if Facet is not a facet or it is a volatile-qualified facet. [ edit ] Notes std::has_facet must return true for all locales loc if Facet is one of the standard facets given here .
