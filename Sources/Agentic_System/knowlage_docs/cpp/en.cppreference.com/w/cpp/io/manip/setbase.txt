get_time (C++11) put_money (C++11) put_time (C++11) Quoted manipulator quoted (C++14) [edit] Defined in header <iomanip> /*unspecified*/ setbase ( int base ) ; Sets the numeric base of the stream. When used in an expression out << setbase ( base ) or in >> setbase ( base ) , changes the basefield flag of the stream out or in , depending on the value of base : the value 16 sets basefield to std::ios_base::hex . the value 8 sets std::ios_base::oct . the value 10 sets std::ios_base::dec . Values of base other than 8, 10, or 16 reset basefield to zero, which corresponds to decimal output and prefix-dependent input.
