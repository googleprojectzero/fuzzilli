the same as another type (concept) [edit] derived_from (C++20) specifies that a type is derived from another type (concept) [edit] convertible_to (C++20) specifies that a type is implicitly convertible to another type (concept) [edit] common_reference_with (C++20) specifies that two types share a common reference type (concept) [edit] common_with (C++20) specifies that two types share a common type (concept) [edit] integral (C++20) specifies that a type is an integral type (concept) [edit] signed_integral (C++20) specifies that a type is an integral type that is signed (concept) [edit] unsigned_integral (C++20) specifies that a type is an integral type that is unsigned (concept) [edit] floating_point (C++20) specifies that a type is a floating-point type (concept) [edit] assignable_from (C++20) specifies that a type is assignable from another type (concept) [edit] swappable swappable_with (C++20) specifies that a type can be swapped or that two types can be swapped with each other (concept) [edit] destructible (C++20) specifies that an object of the type can be destroyed (concept) [edit] constructible_from (C++20) specifies that a variable of the type can be constructed from or bound to a set of argument types (concept) [edit] default_initializable (C++20) specifies that an object of a type can be default constructed (concept) [edit] move_constructible (C++20) specifies that an object of a type can be move constructed (concept) [edit] copy_constructible (C++20) specifies that an object of a type can be copy constructed and move constructed (concept) [edit] Comparison concepts equality_comparable equality_comparable_with (C++20) specifies that operator == is an equivalence relation (concept) [edit] totally_ordered totally_ordered_with (C++20) specifies that the comparison operators on the type yield a total order (concept) [edit] Object concepts movable (C++20) specifies that an object of a type can be moved and swapped (concept) [edit] copyable (C++20) specifies that an object of a type can be copied, moved, and swapped (concept) [edit] semiregular (C++20) specifies that an object of a type can be copied, moved, swapped, and default constructed (concept) [edit] regular (C++20) specifies that a type is regular, that is, it is both semiregular and equality_comparable (concept) [edit] Callable concepts invocable regular_invocable (C++20) specifies that a callable type can be invoked with a given set of argument types (concept) [edit] predicate (C++20) specifies that a callable type is a Boolean predicate (concept) [edit] relation (C++20) specifies that a callable type is a binary relation (concept) [edit] equivalence_relation (C++20) specifies that a relation imposes an equivalence relation (concept) [edit] strict_weak_order (C++20) specifies that a relation imposes a strict weak ordering (concept) [edit] Customization point objects ranges::swap (C++20) swaps the values of two objects (customization
