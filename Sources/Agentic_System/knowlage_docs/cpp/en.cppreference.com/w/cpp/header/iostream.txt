the Input/output library. Including <iostream> behaves as if it defines a static storage duration object of type std::ios_base::Init , whose constructor initializes the standard stream objects if it is the first std::ios_base::Init object to be constructed, and whose destructor flushes those objects (except for cin and wcin ) if it is the last std::ios_base::Init object to be destroyed.
