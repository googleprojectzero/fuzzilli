(C++11) BitmaskType LiteralType (C++11) NumericType UnaryTypeTrait (C++11) BinaryTypeTrait (C++11) TransformationTrait (C++11) Clock (C++11) TrivialClock (C++11) [edit] The named requirements listed on this page are the named requirements used in the normative text of the C++ standard to define the expectations of the standard library. The burden to ensure that library templates are instantiated with template arguments that satisfy these requirements is on the programmer. Failure to do so may result in very complex compiler diagnostics. Some of these requirements are formalized in C++20 using the concepts language feature.
