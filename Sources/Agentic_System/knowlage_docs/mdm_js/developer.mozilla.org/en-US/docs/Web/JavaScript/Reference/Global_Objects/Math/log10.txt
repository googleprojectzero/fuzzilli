Description Because log10() is a static method of Math , you always use it as Math.log10() , rather than as a method of a Math object you created ( Math is not a constructor). This function is the equivalent of Math.log(x) / Math.log(10) . For log10(e) , use the constant Math.LOG10E , which is 1 / Math.LN10 .
